# Security Policy

## Reporting a Vulnerability

Please do **not** open a public GitHub issue for security vulnerabilities.

Report vulnerabilities privately via GitHub's security advisory system:

**[Report a vulnerability](https://github.com/FelipeFuhr/ffreis-platform-cli/security/advisories/new)**

You will receive a response within 5 business days. If the issue is confirmed,
a fix will be prepared and released as soon as possible.
